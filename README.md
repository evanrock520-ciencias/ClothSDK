ClothSDK is a C++ cloth simulation SDK designed to integrate into Digital Content
Creation (DCC) tools such as Blender.

## How to Build

```bash
git clone https://github.com/evanrock520-ciencias/ClothSDK.git
cd ClothSDK
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make
```

## How to Run

If you want to run the python scripting example.

```bash
python3 examples/simulation.py
```
